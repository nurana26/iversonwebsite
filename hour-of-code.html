Hour of Code Event

Empowering students to take their first steps into the world of programming.

About the Event

The Hour of Code is a global movement introducing students to the basics of programming. Our event, designed specifically for 6th-grade students, provided an engaging introduction to coding concepts through interactive activities and problem-solving exercises.

  • Date: November 5, 2024
  • Location: Turkish Diyanet Foundation Baku Turkish Lyceum
  • Participants: 24 enthusiastic 6th-grade students

Event Photos

Event Highlights

During the event, students

  1. Learned basic programming concepts such as loops, conditions, and variables.
  2. Completed a hands-on project: a simple game developed using block-based coding tools.
  3. Received certificates of participation for their efforts.
